SQL 命令

基础命令

  • select
  • select distinct
    去重
  • where
    过滤
  • and or
  • order by
    排序 默认为升序排序，若需要降序则使用关键字DESC
  • insert into
    插入
  • update
    修改
  • delete
    删除

    进阶命令

  • select top
    top, limit, rownum
  • like
    模糊搜索
通配符 说明
% 代表零个、一个或者多个任意的字符
_ 代表单个字符或者数字
[charlist] 字符列表中的任何单一字符。可以使用连字符（-）根据 ASCII 编码指定一个字符范围，例如：
[0-9] 表示从 0 到 9 的任一数字
[a-z] 表示小写英文字母
[a-zA-Z] 表示英文字母，不区分大小写
[a-zA-Z0-9] 表示英文字母和阿拉伯数字
[^charlist] 或 [!charlist] 不在字符列表中的任何单一字符
  • in
    配合where子句使用
  • between
    选取介于两个值之间的数据范围内的值。
  • 别名 as
    通常配和SQL函数使用。

    join

join类型 sql
内连接 sql inner join
全连接 sql full join
左连接 sql left join
右连接 sql right join

  • inner join
    inner join 与 join 相同
  • full join
  • left join
  • right join
  • union
    合并多个select语句的结果, union将选取不同的值。union all允许重复的值。

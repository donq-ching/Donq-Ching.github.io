Fastapi Blog Api Docs

Author: zoyass Date: 三月 20, 2023  10:28:41

项目地址：https://github.com/zoyassisoe/fastapi-blog

统一

  • 若请求api成功则状态码（status）统一为200。
  • 若请求api逻辑成功（例如请求login，且登录成功），在response data信息中code码为200或2000+x（x=1,2,3…）。
  • 若请求api逻辑失败（例如请求login，且登录失败），在response data信息中code则有相应的错误码对应，且携带message进行说明。
